LinkedIn Profile Scraper

Extract posts from LinkedIn profiles

Enter the full URL of the LinkedIn profile you want to scrape.
Enter the maximum number of posts you want to extract.

How it works:

  1. Enter the LinkedIn profile URL you want to scrape
  2. A new browser window will open for LinkedIn
  3. If you're already logged in to LinkedIn, scraping will begin automatically
  4. If not logged in, you'll need to log in first (supports two-factor authentication)
  5. After login, the scraper will automatically navigate to the profile and extract posts
  6. Results will be available to download as a CSV file
For best results: Start Chrome or Edge with remote debugging enabled.

For Chrome: Start with --remote-debugging-port=9222

For Edge: Start with --remote-debugging-port=9222

Example (macOS): /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222
